# tictactoe
ROAM TicTacToe Test
# Built With
 * Laravel 5.5.44 (Mac OSX - Sierra) - API
 * ReactJS - Web App
# Installation
 * before your Laravel migrate script, run MySQL > tictactoe_table.sql in mysql - this creates the database and the user needed for the project to set up and run.
# Execute
 * start up your Laravel API - you can check the GET endpoint http://tictactoe.api/api/games (you should get an empty JSON data array)
 * Start up your React Web App in the folder web_app
